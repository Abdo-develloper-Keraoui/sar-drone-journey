# sar-drone-journey

Teaching myself the real foundations of drone software — starting from scratch, in public, one day at a time.

---

Drone research in Morocco is emerging, but there's still a lot of ground to cover. Drones aren't a niche thing anymore — they're becoming the workhorse of everything: farming, delivery, search and rescue, infrastructure. I want to be someone who helps build that.

So this is me building the foundation properly. Not tutorials. The actual stuff — C, linear algebra, probability, control theory, sensor fusion. In order, for a reason.

---

This repo is part learning log, part paper trail, part promise to myself.

`daily-log.md` gets a commit every day. `month-01/` is where I am right now, week one.

That's it. Still early. Just started.
